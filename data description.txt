All the original data files are stored in the folder data. There are two files-21-23hr.csv and ballparks.csv.
21-23hr.csv contains all the number of HR in every stadiums in MLB from 2021 to 2023 seasons. These data are collected from https://www.onlyhomers.com/daily.
ballparks.csv contains all the MLB stadiums information. These data are from kaggle.
Both these two files' data are in correct form and types. And there is no missing value.
